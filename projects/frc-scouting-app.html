Project Details

QSTAR provides AI-powered strategy analysis for FRC teams, helping them make data-driven decisions. Key features include:

  • Real-time team performance analysis
  • AI-powered strategy recommendations
  • EPA data integration
  • Detailed strategy breakdowns for Auto, Teleop, and Endgame periods
  • Interactive team selection interface

Built with: React + Vite, Node.js + Express, Groq LLM API, Statbotics API
