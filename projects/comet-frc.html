Project Details

Comet FRC is a Java-based control system designed for the Marina robot used in FRC competitions. It supports real-time motor control and sensor feedback to make autonomous decisions.

  • Motor control algorithms
  • Sensor data integration
  • Real-time feedback loop for autonomous behavior
