Project Details

Marina FRC is a Java-based control system designed for our FRC competition robot. It features advanced autonomous capabilities and precise motor control systems.

  • Advanced autonomous pathing to specific nodes on the reef and barge shooting locations
  • Full simulation capabilities using AdvantageScope and the AdvantageKit framework
  • Precise subsystem control using PID controllers with Feed Forward compensation
  • Motor control algorithms
  • Sensor data integration
  • Real-time feedback loop for autonomous behavior

Built with: Java, WPILIB, PathPlanner, and FRC Libraries
